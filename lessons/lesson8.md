## Lesson 8: Dashboards in Yandex Datalens

## 1. What is DataLens

Yandex DataLens is a business analytics tool for data visualization and dashboards. It allows you to connect data sources, build charts, and create interactive dashboards for analysis and reporting.

## 2. Main Objects

* **Connection** – link to a data source
* **Dataset** – structured data and its schema
* **Chart** – data visualization (table, graph, etc.)
* **Dashboard** – collection of charts and widgets

A dataset defines data structure and fields used for analysis.

## 3. Basic Workflow

1. Create **Connection**
2. Create **Dataset**
3. Build **Charts**
4. Combine into **Dashboard**

This is the standard process described in the official quick start.

## 4. Charts

A chart is a visualization of data (table, diagram, or map).
Charts are created by selecting a dataset and dragging fields into axes.

Common types of charts:

* Bar chart – category comparison
* Line chart – trends over time
* Table – detailed data view

## 5. Dimensions and Measures

* **Dimension** – categorical variable (e.g., country, device)
* **Measure** – numeric metric (e.g., revenue, count)

These define how data is grouped and aggregated in charts.

## 6. Aggregation and Filters

* Aggregations: sum, average, count
* Filters: restrict data by category or range

Filters can be applied at chart or dashboard level.

## 7. Dashboards

A dashboard is a page (or set of pages) that contains charts and other widgets grouped together.

It allows:

* combining multiple visualizations
* adding filters (selectors)
* presenting results in one view

## 8. Creating a Dashboard

1. Click **Create dashboard**
2. Add charts as widgets
3. Arrange layout
4. Save

Dashboards combine charts into a single analytical view.
